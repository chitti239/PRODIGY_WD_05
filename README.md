# 🌦 WeatherSphere

WeatherSphere is a modern, responsive weather dashboard built using **HTML**, **CSS**, and **JavaScript**, powered by the OpenWeatherMap API.

It provides real-time weather information, air quality data, location-based forecasts, and dynamic weather-themed UI backgrounds that change according to current weather conditions.

---

## ✨ Features

### 🌍 Real-Time Weather

* Search weather by city name
* Current location weather using Geolocation API
* Real-time temperature updates
* Feels-like temperature
* Weather description and icon

### 📊 Weather Details

* Humidity
* Wind Speed
* Atmospheric Pressure
* Visibility
* Sunrise Time
* Sunset Time

### 🌱 Air Quality Index (AQI)

* Live AQI data
* AQI categories:

  * Good
  * Fair
  * Moderate
  * Poor
  * Very Poor

### 📅 5-Day Forecast

* Daily weather predictions
* Weather icons
* Temperature forecasts
* Responsive forecast cards

### 🎨 Dynamic Weather Themes

The application's background automatically changes based on weather conditions:

| Weather        | Theme  |
| -------------- | ------ |
| Clear Sky      | Sunny  |
| Clouds         | Cloudy |
| Rain / Drizzle | Rainy  |
| Thunderstorm   | Storm  |
| Snow           | Snow   |
| Night Time     | Night  |

### ✨ Modern UI

* Glassmorphism cards
* Floating animated background effects
* Neon gradients
* Smooth hover animations
* Mobile-first responsive design
* Modern dashboard appearance

